Pizza Module

class src.pizza.Pizza(crust, sauce, cheese, toppings)[source]

Bases: object

Pizza class for creating and managing pizza objects.

This class handles the creation of pizza objects with various ingredients and calculates the cost based on the selected options.

crust

Type of crust (thin, thick, or gluten_free)

Type:

str

sauce

List of sauces (marinara, pesto, liv_sauce)

Type:

list

cheese

Type of cheese (only mozzarella supported)

Type:

str

toppings

List of toppings (pineapple, pepperoni, mushrooms)

Type:

list

Example

>>> pizza = Pizza("thin", ["marinara"], "mozzarella", ["pepperoni"])
>>> print(pizza)
Thin crust pizza with marinara sauce, mozzarella cheese, and pepperoni. Cost: $9

cost()[source]

Calculate the cost of the pizza based on its ingredients.

The cost is calculated by adding: - The cost of the crust - The cost of each sauce - The cost of each topping

Returns:

Total cost of the pizza

Return type:

int

Example

>>> pizza = Pizza("thin", ["marinara"], "mozzarella", ["pepperoni"])
>>> pizza.cost()
9
