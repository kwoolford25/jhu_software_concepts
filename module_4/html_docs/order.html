Order Module

class src.order.Order[source]

Bases: object

Order class for managing customer pizza orders.

This class handles the creation and management of customer orders, including adding pizzas, calculating total cost, and processing payment.

pizzas

List of Pizza objects in the order

Type:

list

cost

Total cost of the order

Type:

int

paid

Payment status of the order

Type:

bool

Example

>>> order = Order()
>>> order.input_pizza("thin", ["marinara"], "mozzarella", ["pepperoni"])
>>> print(order)
Your order:
1. Thin crust pizza with marinara sauce, mozzarella cheese, and pepperoni. Cost: $9

Total cost: $9 (NOT PAID)

__init__()[source]

Initialize a customer order with empty pizza list and zero cost.

The order starts with no pizzas, zero cost, and is marked as not paid.

Example

>>> order = Order()
>>> len(order.pizzas)
0
>>> order.cost
0
>>> order.paid
False

input_pizza(crust, sauce, cheese, toppings)[source]

Add a pizza to the order.

This method creates a new Pizza object with the specified ingredients and adds it to the order, updating the total cost.

Parameters:

  • crust (str) – Type of crust

  • sauce (list) – List of sauces

  • cheese (str) – Type of cheese

  • toppings (list) – List of toppings

Example

>>> order = Order()
>>> order.input_pizza("thin", ["marinara"], "mozzarella", ["pepperoni"])
>>> len(order.pizzas)
1
>>> order.cost
9
order_paid()[source]

Mark the order as paid.

This method updates the payment status of the order to indicate that payment has been received.

Example

>>> order = Order()
>>> order.input_pizza("thin", ["marinara"], "mozzarella", ["pepperoni"])
>>> order.paid
False
>>> order.order_paid()
>>> order.paid
True
